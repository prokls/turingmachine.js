turingmachine.js
================

:release:   0.9.0-unstable
:date:      October 2013 - Nov 2014
:status:    basic usecases work
:author:    meisterluk
:license:   CC0

A turingmachine for educational purposes.

Installation
------------

Just open ``index.html`` in some modern browser.
